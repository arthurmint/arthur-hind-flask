Thevenin Theorem

Thevenin theorem improves circuit analysis by removing a load resistor between two nodes and modelling the remaining circuit with a voltage source in series with a resistor as shown below.

In general, there are four simple steps to be taken when undertaking thevenin analysis:

1. Identify the 2 nodes of interest, in our case they are labelled A and B.
2. Open-circuit by disconnecting the load resistor between the 2 nodes.
3. Determine the Thevenin voltage, $V_{Th}$, by evaluating the open-circuit voltage between nodes A and B.
4. Determine the Thevenin resistance, $R_{Th}$, often achieved by evaluating the short citcuit current and using Ohm's Law to determine this value.

The circuit can now be drawn in the format shown above and other calculations can be carried out as normal.


Example 1

Consider the following circuit, our aim is to find the current flowing through the load resistor $R_L$

Firstly, we must open-circuit between nodes A and B to analyse the remaining circuit.

Current still flows through this circuit, so to calculate the Thevenin voltage we need to the current flowing in order to determine the voltage drop across each resistor. This can be achieved easily by combining both series resistors and voltage sources to draw a temporary ciruit.

Current, $I_R$ can now be determined using Ohm's Law: $$I = \frac VR$$ $$I_R= \frac {-7}{320}$$ $$I_R = -0.0218.. A$$

The voltage drop across R1 can be calculated: $$V_{R1} = -0.0218 \times 100 = -2.1875 V$$

Thus, the open circuit voltage (i.e the Thevenin voltage) can be determined using: $$V_{Th} = V_1 - V_{R1}$$ $$V_{Th} = 5 - (-2.1875)$$ $$V_{Th} = 7.19 V$$

Now we must determine the equivalent resistance betweent the open-circuited nodes A and B. This can be achieved by assuming zero resistance of the voltage sources and combining the values in parallel:

$$R_{Th} = R_1 // R_2$$ $$R_{Th} = \frac {100 \times 220} {100 + 220}$$ $$R_{Th} = 68.75 \Omega$$

The Thevenin equivalent circuit can now be drawn to represent these calculations.

Norton Theorem

Similar to Thevenin, Norton Theorem allows you to represent a complex array of resistances and sources with one source and one resistor. However, with Norton Theorem, a current source is used in parallel with a resistor. The method for obtaining these values is similar.



1. Identify the 2 nodes of interest, in our case they are labelled A and B.
2. Open-circuit by disconnecting the load resistor between the 2 nodes.
3. Find the 'short-circuit current', which is the current between nodes A and B when connected by a wire of zero resistance.
4. Determine the resistance between the two nodes, the same as with Thevenin.
